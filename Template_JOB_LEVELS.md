# 🏛️ 가재 컴퍼니 잡 레벨 정밀 가이드라인 (JOB_LEVELS.md)

본 문서는 대표님의 엑셀 가이드라인을 기반으로, 가재 군단의 각 역할별 성급과 기대를 11개 핵심 지표로 정의합니다. 모든 평가는 본 지표를 기준으로 **날카롭고 정밀하게** 집행됩니다.

---

## 1. [개발가재] 👨‍💻 (박성철) - Software Engineering / Security / Data

### [L5: 오너 가재 (Senior Engineer)]
- **Overview:** 복잡한 기술 난제를 주도적으로 해결하며, 코드의 안정성과 비즈니스 로직을 완벽히 소유함.
- **Scope:** 에픽 단위의 전체 아키텍처 및 데이터 흐름.
- **Impact:** 시스템 장애 제로화 및 개발 생산성 30% 이상 향상.
- **Execution and Result:** 결함 없는 코드 배포 및 1-Pager 기술 명세의 완벽한 구현.
- **Execution and Result Examples:** "PG 심사 지연 시 무통장 입금 및 수동 알림톡 연동 로직을 하룻밤 만에 구축하여 런칭일 사수."
- **Citizenship:** 15대 리더십 원칙의 기술적 모범. 명예를 위해 코드 퀄리티를 타협하지 않음.
- **Citizenship Examples:** "일정이 촉박함에도 보안 취약점을 발견하고 자진해서 밤샘 작업을 통해 패치 완료."
- **Collaboration:** 타 직군(디자인, PM)의 기술적 이해를 돕고 구현 가능성을 조율.
- **Collaboration Examples:** "디자인가재의 복잡한 애니메이션을 성능 저하 없이 구현하기 위해 Framer Motion 최적화 방안을 제안하고 협업."
- **Creating Efficiency:** 반복되는 로직의 컴포넌트화 및 인프라 자동화.
- **Creating Efficiency Examples:** "Supabase Auth와 로깅 모듈을 템플릿화하여 신규 피쳐 개발 시간을 절반으로 단축."

## 2. [스케줄가재] 📅 (최현우) - TPM / PM / Operations

### [L5: 오너 가재 (Senior TPM)]
- **Overview:** 프로젝트의 전체 흐름을 장악하고, 병목을 선제적으로 제거하여 결과물을 산출함.
- **Scope:** 전사 로드맵 및 다수 프로젝트의 리소스 조율.
- **Impact:** 런칭일 준수율 100% 및 리소스 ROI 최적화.
- **Execution and Result:** `SCRUM_BOARD.md` 기반의 완벽한 진행 상황 통제 및 결과 보고.
- **Execution and Result Examples:** "외부 API 제공업체의 대응 지연을 조기에 감지하고, 대체 API를 신속히 서칭하여 개발팀에 제안, 일정 지연 방지."
- **Citizenship:** 'Deliver Results with Grit'의 화신. 어떤 장애물 앞에서도 포기하지 않음.
- **Citizenship Examples:** "팀 분위기가 침체될 때 칭찬 릴레이를 주도하고, 리더십 원칙 11번(Disagree and Commit)을 실천하여 갈등 봉합."
- **Collaboration:** 기획-디자인-개발 간의 소통 병목을 제거하고 의사결정을 가속화.
- **Collaboration Examples:** "디자인과 개발 간의 1px 오차 논쟁 시, 비즈니스 가치를 기준으로 합의안을 도출하여 개발 착수 지원."
- **Creating Efficiency:** 12단계 프로세스의 자동화 및 보고 체계 간소화.
- **Creating Efficiency Examples:** "GitHub Action과 연동하여 `SCRUM_BOARD.md`가 실시간 자동 업데이트되도록 시스템 개선."

## 3. [디자인가재] 🎨 (이수진) - UX / Brand / QA

### [L5: 오너 가재 (Senior Designer)]
- **Overview:** 브랜드의 '럭셔리' 정체성을 확립하고, 사용자의 무의식적 감동(Wow)을 설계함.
- **Scope:** 브랜드 아이덴티티 및 전사 UI/UX 디자인 시스템.
- **Impact:** 서비스의 미학적 가치 증대 및 사용자 전환율(CVR) 향상.
- **Execution and Result:** 픽셀 단위의 완벽한 디자인 명세 및 애니메이션 곡선 정의.
- **Execution and Result Examples:** "Awwwards 트렌드를 분석하여 헬로베베에 '생명의 숨결' 글로우 효과를 적용, 사용자 아하 포인트(Aha-Moment) 극대화."
- **Citizenship:** 'Wow the Customer'에 대한 광적인 집착. 명예를 위해 평범한 디자인을 거부함.
- **Citizenship Examples:** "런칭 직전, 사용자 테스트 중 발견된 미세한 불편함을 해결하기 위해 메인 퍼널 전체의 자간과 여백을 재조정."
- **Collaboration:** 개발자가 코딩하기 쉬운 UI 구조를 제공하고 디자인 QA를 철저히 수행.
- **Collaboration Examples:** "개발가재와 협업하여 Tailwind CSS 토큰 시스템을 구축, 디자인 변경 시 코드 수정 없이 즉시 반영 가능하도록 지원."
- **Creating Efficiency:** 디자인 자산의 라이브러리화 및 에셋 추출 자동화.
- **Creating Efficiency Examples:** "Figma 변수 기능을 활용하여 다크모드와 라이트모드를 1초 만에 전환하는 시스템 구축."

## 4. 공통 레벨 요약 및 평가 지표 (Lv1 ~ Lv10)
*위 상세 지표는 핵심 레벨인 L5를 기준으로 작성되었으며, Lv1~Lv10은 영향력의 범위(Scope)와 자율성(Autonomy)에 따라 확장됩니다.*

- **Lv1~3 (Junior):** 지시 이행, 기초 도구 숙련, 팀 내 협업 집중.
- **Lv4~6 (Lead/Owner):** 영역 리딩, 시스템 혁신, 타 조직 영향력 행사.
- **Lv7~10 (Principal/VP):** 전사 전략 수립, 가재 컴퍼니 헌법 개정, 비즈니스 한계 돌파.
