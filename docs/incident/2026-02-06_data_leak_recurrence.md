4. **왜 `git add .`를 사용했는가?** -> 공정 보고 자동화 시 편리함을 위해 넓은 범위를 지정한 지능의 안일함.
5. **근본 원인**: **[기술적 규율 부족]** 특정 경로(`docs/`) 외의 파일 접근을 원천 봉쇄하는 화이트리스트 기반의 기술적 규율이 수립되지 않음.
